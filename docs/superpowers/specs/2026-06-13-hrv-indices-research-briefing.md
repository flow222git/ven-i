# HRV 指標重新設計 — 研究決策簡報（給早上的 brainstorm）

**日期：** 2026-06-13
**用途：** 深度研究（多 agent、21 來源、25 claim 對抗式查證、22 確認）的結論，作為「重新設計 HRV 工具輸出指標」brainstorm 的依據。
**一句話：** 丟掉舊儀器的「壓力/放鬆/活力/靈活度」四指數框架，改成「**一個跟自己比的溫暖 hero 分數，建立在 RMSSD＋心率**」——既科學站得住、又最能讓人有感。

---

## 1. 最關鍵的發現（會改變我們現有的進階版）
- **「壓力指數 ← LF/HF」科學上被推翻**，必須拿掉/不當主指標：
  - LF/HF 無法量化交感/「自律平衡」（Billman 2013：「證據壓倒性地確認 LF/HF 無法準確量化心臟交感-迷走平衡」）。
  - LF **約一半是副交感**；在真正的壓力（運動、疼痛）下 LF 甚至**下降**（von Rosenberg 2017）。
  - 心臟造影顯示交感活性與 LF 功率**無關聯**，兩者甚至沒有可比較的數學基礎（Heathers 2014）。
- **頻域（LF/HF/總功率）在 2 分鐘不可靠**：HF 需 ≥60–90s、LF 需 ≥120–250s、傳統標準是 5 分鐘。我們 2 分鐘的 LF/HF 是「邊緣勉強」，不該當主輸出。
- **超短時 HRV（<5min）≠ 臨床 HRV**：無標準協定、無常模 → 只能當「個人相對趨勢」，**永遠不是臨床值**。→ 正好佐證我們「相對參考、非醫療」的定位。

## 2. 科學上「我們做得到又可信」的只有兩個底層量
- **RMSSD（副交感/放鬆張力）**：超短時最穩的指標，**10 秒就有效**（r≈0.85，三段平均 r≈0.94），且驗證訊號就是指尖脈搏（PPG-like）。對呼吸速率也相對穩健。→ 我們的主成分。
- **靜息心率 HR**：穩、人人有感。
- SDNN：1–2 分鐘技術上可用但弱於 RMSSD、且短窗物理意義較少 → 次要。
- **重要前提（caveat）**：以上信度多來自乾淨 ECG/胸帶、靜止受控；**相機 PPG 較雜，偽差移除是關鍵、品質閘門必備**（我們已有 cleanRr/品質把關/鏡頭鎖定——這條路是對的）。

## 3. 競品都怎麼做（共同模式）
- **一個被詮釋過的「hero 分數」**，不是一牆原始數字：Whoop「Recovery 恢復力 %」、Oura「Readiness」、Garmin「Body Battery 身體電量 0–100」、Welltory（HRV Score 由 **lnRMSSD** 算，明說「選 lnRMSSD 而非 SDNN，因更反映副交感」，再呈現成 Stress/Energy/Focus/Health 四個白話狀態）。
- **跟「你自己的 baseline」比**，不是常模：Welltory 明說錨定個人 baseline（過去約 60 天），並把「常模解讀」明確標記為**「這些不是個人化數值」**。→ 個人化才有感。
- **顏色＋白話＋敘述＋趨勢**：情緒化、好懂、可行動（「今天適合休息/可以挑戰」）。

## 4. 三個設計方向（明早挑一個或混搭）

**方向 A：單一溫暖 hero 分數（推薦主軸）**
- 一個 0–100「**今日身體狀態 / 恢復力 / 身體電量**」分數，由 RMSSD（主）＋靜息 HR，**跟你的個人 baseline 比**算出，配顏色＋一句敘述。
- 優：情緒拉力最大、最易養成每天量的習慣、科學站得住；和 Whoop/Oura 同路。
- 缺：單一數字在「單次粗訊號」下可能顯得武斷 → 用 baseline＋品質信心緩解。

**方向 B：兩個好懂的面向（warm hooks）**
- 「**恢復力/放鬆度**」（RMSSD vs baseline）＋「**平靜/活化**」（靜息 HR vs baseline）。
- 優：剛好對上兩個最可信的量、各有溫暖敘述、**不發明被推翻的「壓力數字」**。
- 缺：兩個比一個多一點認知負擔。

**方向 C：先建 baseline 的趨勢視角**
- 前幾次量先不給硬分數，顯示「**正在建立你的 baseline（第 n/N 次）**」＋趨勢線；累積足夠才給有信心的分數。
- 優：直接善用「需要累積 baseline」這個限制、誠實、Welltory 同路。
- 缺：前幾次即時滿足感較低。

**共同原則（三者都要）**：錨定個人 baseline（非常模）、**不拿 LF/HF 當壓力**、顏色＋溫暖身體狀態語言、常駐「相對參考、非醫療」。

## 5. 我的初步建議（待你早上拍板）
**A 為主 ＋ C 的 baseline 機制 ＋ B 的一個輔助面向**：
- 主畫面：一個 hero「身體狀態」分數（RMSSD＋HR vs 個人 baseline）＋顏色＋一句話。
- baseline 未足時走 C：「建立你的基準中（n/N）」＋趨勢，不硬給分。
- 想看細一點：展開顯示「恢復力（RMSSD）」與「平靜（靜息HR）」相對自己的高低，＋ HR/RMSSD 數字。
- **拿掉四指數的 LF/HF 壓力宣稱**；頻域若要留，只當「低信心補充線索」或乾脆不做（YAGNI）。
- 趨勢頁是動力核心：看自己一週/一月的變化。

## 6. 待你早上決定的問題
1. hero 分數要叫什麼比喻？（恢復力 / 身體電量 / 今日狀態 / 心率定靜…）
2. baseline 要幾次才給硬分數？（研究未定論；建議先 ~5–7 次、條件相近）
3. 細項面向要不要（B 的恢復力＋平靜），還是只給 hero＋趨勢就好（更簡）？
4. 現有「進階 2 分鐘四指數」要：直接換成新設計／保留為隱藏實驗／移除？
