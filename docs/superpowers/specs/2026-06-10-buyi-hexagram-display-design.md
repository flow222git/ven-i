# 卜易結果頁 · 卦象視覺化改版設計

## 背景
目前卜易結果頁的頂部 hex-card 只顯示卦名、卦類（如「天天卦」）、與一句卦義，沒有六爻圖。六爻圖被放在下面的「你擲出的卦（由下而上）」面板裡，與本卦解讀資訊脫節，使用者看到結果第一眼缺乏直觀的卦象視覺。

## 目標
讓使用者在結果頁第一眼就同時看到：
1. 卦象六爻圖
2. 上卦／下卦標示（含八卦名與自然象）
3. 本卦的三點核心重點

全部整合在頂部的彩色 hex-card 裡，成為一個「一張卡看完本卦」的主視覺。

## 設計

### 改動位置
- `index.html` 的 `buyiResult()` 函數（約 1245-1344 行）
- 重寫頂部 hex-card 內部結構
- 移除「你擲出的卦（由下而上）」整個 panel（連同其下的逐爻意義列表）

### Hex-card 新版結構

```
┌────────────────────────────────────────┐
│ 本卦・你目前的處境                       │
│                                        │
│  乾                ▬▬▬▬▬   6 ─┐        │
│  天天卦            ▬▬▬▬▬   5 ─┤ 上卦   │
│                    ▬▬▬▬▬   4 ─┘ 乾・天 │
│                    ─ ─ ─ ─ ─ ─         │
│                    ▬▬▬▬▬   3 ─┐        │
│                    ▬▬▬▬▬   2 ─┤ 下卦   │
│                    ▬▬▬▬▬   1 ─┘ 乾・天 │
│                                        │
│  純粹的創造與剛健，適合開創，但忌過剛獨斷。 │
│  ────────────────────────────────       │
│  本卦重點                               │
│    • 核心訊息：純粹的創造與剛健          │
│    • 此刻態度：保持正氣，不獨斷          │
│    • 避免的事：過剛、強逞                │
└────────────────────────────────────────┘
```

### 視覺細節
- **六爻圖**：陽爻=整條、陰爻=斷開（保留現有視覺邏輯）
- **顏色**：在彩色 hex-card 底色上，爻線改用白色 / 半透明白色，與底色對比
- **變爻標記**：保留 ○（老陰）/ ✕（老陽），放在爻號旁
- **上下卦分隔線**：第 3 爻與第 4 爻間一條淡色虛線
- **上下卦標籤**：以括號連到上半 3 爻 / 下半 3 爻，標示「上卦」「下卦」+ 八卦名 + 自然象（如「乾・天」）
- **本卦重點**：以 ─── 分隔線與卦義隔開，三點 bullet，標籤化短句

### 新增資料結構

```js
const GUA_KEY = {
  乾: { core: "純粹的創造與剛健", attitude: "保持正氣，不獨斷", avoid: "過剛、強逞" },
  坤: { core: "...", attitude: "...", avoid: "..." },
  // ...64 卦
};
```

文案策略：
- 以現有 `GUA_MEAN` 為骨架，拆分為三欄
- 「核心訊息」≈ 描述當下處境（從卦義抽取）
- 「此刻態度」≈ 建議的應對心態（從卦義抽取或補寫）
- 「避免的事」≈ 警示與禁忌（從卦義抽取或補寫）
- 維持與 app 整體一致的生活化、不命定語氣

### 變卦與其他面板
- **變卦面板**：保留現狀（這一張新卡只處理本卦）
- **焦點面板**（這一卦重點看哪裡）：保留
- **領域提點、總結、免責**：保留

## 不做的事（避免範圍蔓延）
- 不改 hex-card 的底色三態邏輯（仍是 b-good / b-neutral / b-lack）
- 不改變卦的顯示
- 不改問易（wenyi）模式
